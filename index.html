    Why Not Just Use Bootstrap
    I love Bootstrap. I do. But it is large and I do not like **all** that grid business when CSS Grid is **so** much more efficient and elegant.
    Bootstrap Forms
    All credit to bootstrap for the CSS for the forms elements. The CSS file itself credits Bootstrap for the styling of the form elements but beyond that I often use the Bootstrap approach to validation as well.
    This is Not Bootstrap Compatible
    All that being said, this is not a Bootstrap template and is not Bootstrap compatible. I have simply used a few of the items that I feel they do very well and incorporated them into my own CSS Grid based HTML Template.
